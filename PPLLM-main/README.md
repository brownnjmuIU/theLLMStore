PPLLM
=====

This repository combines two related components into a single project:

- **file-aggregation-tool**: A file crawler and aggregation utility (originally from `file-aggregation-tool`).
- **permission_layer**: A lightweight permissions and access logging layer (originally from `permission_layer`).

## Structure

- `file-aggregation-tool/` — Source, docs, and build artifacts for the file aggregation tool.
- `permission_layer/` — Permission management code, configuration, and access log database.
